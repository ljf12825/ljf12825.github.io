优先级（Precedence）与 结合性（Associativity）

优先级决定了哪些运算先执行。优先级高的运算先于优先级低的运算

结合性决定了相同优先级的运算符如何分组

两种结合性

1.左结合（从左到右）：大多数运算符 2.右结合（从右到左）：赋值、单目、三元条件运算符

示例

// 左结合
int a = 10 - 5 - 2;

// 右结合
int x, y, z;
x = y = z = 5;
