Physical Component
Modified: 2025-12-31 | Author：ljf12825
  1. 电压、电流
  • 电压（Voltage）：不同电平表示不同的逻辑状态，比如高电平 = 1，低电平 = 0
  • 电流（Current）：电荷流动，是电压在电路中产生作用的载体
  • 应用：CPU寄存器、总线传输、逻辑门输入输出

本质：用电信号承载二进制信息

  1. 光信号
  • 光通信：光纤传输使用光脉冲表示0/1，速度快、干扰少
  • 应用：互联网主干网络、数据中心互联

本质：用光的存在/缺失、相位、强度表示信息

  1. 磁性
  • 硬盘存储：用磁化方向（顺/你）表示0/1
  • 磁带：类似原理，用磁带上的磁化区域存信息
  • 磁存储器件：如MRAM

本质：信息通过物质状态的变化存储

  1. 半导体器件
  • 晶体管：最小开关单元，实现电流通/断 -> 逻辑门
  • 二极管、MOSFET：用作开关、信号控制
  • 集成电路：把成千上万晶体管集成在芯片上，实现复杂运算

本质：用微观物理器件控制电流流向，从而实现逻辑运算

  1. 二进制的物理表示
  • 逻辑电平：高电平/低电平对应1/0
  • 编码方式：例如TTL电平、CMOS电平、正负电压编码
  • 多值逻辑（少用）：理论上可以用三值或四值逻辑表示信息，但实际应用少

本质：把信息抽象成最小单位“位（bit）”，映射到物理状态

  1. 存储介质
  • DRAM：电容存储电荷，周期刷新，易失性
  • SRAM：晶体管存储，无需刷新，速度快，但成本高
  • 硬盘：磁性存储，非易失性
  • 闪存：电荷存储在浮栅晶体管上，非易失性
  • 光盘/光存储：通过物理凹坑和光反射表示0/1

本质：通过物理状态长期或短期保存信息

总结

信息的物理载体就是：信息在物理世界的表达方式，它的功能是把抽象的“0/1”映射到可以操作、传输和存储的物理实体上
