## 基本概念

### 基础语法

```bash
gcc/g++ [选项] 源文件 [选项] [目标文件]
```

### 常用选项

#### 基本选项

| 选项 | 作用 |
| - | - |
| `-o <file>` | 指定输出文件名 |
| `-c` | 只编译不链接，生成目标文件(.o) |
| `-E` | 只进行预处理不编译 |
| `-S` | 编译到汇编语言，不汇编 |
| `-v` | 显示编译过程的详细信息 |
