# Low Code

<!--more-->

- 无代码叙事
  - Twine
  - Bitsy
  - RPG Maker / Wolf RPG Editor
- 物理编程：纸、骰子与桌游
  - 这是游戏的起源形态，也是最高级的游戏设计训练营
  - 开发工具：铅笔、一叠A4纸、一堆骰子
  - 迭代方式：口头规则 -> 发现Bug -> 口头热修(HR, House Rule) -> 形成定稿 -> 上传至Tabletopia或Tabletop Simulator
  - 核心体验：剥离了代码和画面的干扰，机制是否好玩会变得极其赤裸。一个连纸面原型都无聊的游戏，做成3A也救不了
