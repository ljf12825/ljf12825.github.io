# Flashback Core

<!--more-->

- `.swf`
- Adobe Animate
- AS3

Flash游戏不只是技术名词，它背后是一整套**特定的美学、交互逻辑和创作氛围**。这些特质放在今天看，反而成了某种稀缺的体验。

### 1. 视觉上：独特的“矢量橡皮管”美学

这是Flash游戏最直观的辨识度。

- **干净锐利的边缘**：因为是矢量计算而非位图缩放，Flash游戏的线条永远清晰。那个时代的动画师极度依赖**轮廓线**和**纯色块**，形成了一种类似“可动插画”的质感。
- **补间动画的生硬感**：现代游戏角色动作多为逐帧精灵图或骨骼动画，而Flash重度依赖**形状补间**和**缓动曲线**。角色移动时那种“滑步”的惯性感、手臂甩动时变形的夸张感，现在看是粗糙，但当时是独特的动态语言。
- **低多边形与高饱和度**：受限于当时的CPU矢量渲染压力，美术资源极度克制。这催生了像《方块堡垒》那种极简几何风，以及《幻想大战》里高饱和度的日系赛璐璐风格。

### 2. 交互上：鼠标悬浮的触觉感

这是最容易被忽略的一点。Flash是为**鼠标指针**设计的，而非触摸屏或手柄。

- **悬浮状态的反馈**：你的鼠标划过按钮，它会弹跳、发光、发出“啵”的一声。现代手游为了适配触屏，把这种**悬停交互**砍光了，但在Flash游戏里，**把鼠标放在那里就是一种玩法**，比如《梦游先生》里的摸索感。
- **拖拽的物理手感**：因为Flash原生支持拖拽事件，大量游戏依赖**按住并甩出去**的机制（比如《投掷武器》《愤怒的小鸟》原型《碎蛋》）。这种在屏幕上模拟“捏住实物”的鼠标手感，是触屏难以复现的。
- **右键菜单的彩蛋**：这是Flash独有的结界。开发者经常把攻略、小玩笑甚至通关密码藏在右键的“关于”里。

### 3. 创作逻辑上：“低保真”催生的超强核心玩法

当年做Flash的人没有Unity的庞大资源商店，没有AAA级的光影。当**画面拼不过大厂，就只能卷玩法创意**。

- **机制的纯粹性**：拿掉复杂渲染后，Flash游戏往往只有一个极度上瘾的**核心循环**。比如《屋顶狂奔》只有左右键和跳跃；《瘟疫公司》最初在Flash上只是一张极简的地图和几个滚动数字。
- **反叛与恶搞精神**：因为发布成本极低（一个swf文件丢上网址就行），Flash是**作者电影**式的土壤。它允许《变态人生大冒险》这种纯粹为了折磨玩家而生的邪典存在，也允许《The Scale of the Universe》这种纯科普互动艺术。

### 4. 一种消失的“社区氛围”

这是Flash最特别但已经**物理消失**的特质：**即时的网页加载与修改权**。

- 你可以右键**另存为**任何一个游戏，用闪客精灵反编译看它的源码是怎么写的——**这种开源精神是现代App生态不具备的**。
- 不需要下载3GB安装包，点开即玩，加载条走完的那一瞬间就是全部。

### 5. 现在还有人在复现这种“特别”吗？

有，而且成了一个独立游戏细分流派，叫 **“Flashback Core”**。

- **代表作**：《Buckshot Roulette》（恶魔轮盘赌）——故意采用低分辨率、粗糙字体和僵硬的画面缩放，用现代引擎模拟Flash的**不安感**。
- **代表作**：《Pizza Tower》——画风和角色动态完全致敬90年代末的Nickelodeon卡通和Newgrounds时代的狂野Flash动画。
