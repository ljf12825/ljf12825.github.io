# [GNU Binutils](https://www.gnu.org/software/binutils/)

<!--more-->

GNU Binutils is a set of low-level development tools for working with binary files, object files, library files, and assembly code\
It is an important component of the GNU toolchain and is typically used in conjunction with GCC and GDB\
Simply, it is responsible for the stages of program generation that occur after compilation and before execution
